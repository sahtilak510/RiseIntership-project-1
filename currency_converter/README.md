# Currency Converter App

A simple Python application to convert currencies using static exchange rates.

## Features
- CLI version: Run in the terminal
- GUI version: Tkinter-based with dropdowns

## Supported Currencies
- USD
- EUR
- GBP
- JPY
- INR and other also

## How to Run

### CLI Version
```
python cli_converter.py
```

### GUI Version
```
python gui_converter.py
```

## Requirements
- Python 3.x
- Tkinter (usually included with Python)

No external dependencies required.
